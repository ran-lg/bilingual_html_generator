# bilingual_html_generator
A bilingual HTML generator, useful to create eBooks

# Setup
   ```sh
   python -m venv venv
   pip install -r requirements.txt

   ```

# Usage
If your two files are HTML files.
   ```sh
   python main.py file1.html file2.html html
   ```
If your two files are not HTML files.
   ```sh
   python main.py file1.txt file2.txt
   ```
